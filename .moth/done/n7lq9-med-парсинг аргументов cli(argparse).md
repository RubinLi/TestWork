
# Парсинг аргументов cli(argparse)

## Аргументы 

- [x] ( не обязателен ) Можно передать пути к файлам через --files
- [x] ( не обязателен ) Можно указать название отчёта через --report (clickbait)
- [x] Короткие команды 
- [x] Справка
- [ ] Локализация

## Тесты

| Тест                  |      --files        |  --report   |
|---------------------- |:-------------------:|------------:|
| Oпциональный  files   |                     | clickbait   |
| Oпциональный  report  |    test.srv         |             |
| Множественный files   | test1.srv test0.srv | clickbait   |
