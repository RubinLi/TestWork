## cli приложение для обработки csv-файлов с метриками видео на YouTube.

Приложение читает файлы с данными о видео (см. пример ниже) и формирует отчеты.  
Нужно сформировать один отчёт, в котором будут перечислены кликбейтные видео.  
Видео считается кликбейтным если обладает одновременно высоким CTR и низким удержаним. 

### Для отчёта нужны следующие колонки из csv-файла:

- title - название видео
- ctr - click-through rate, кликабельность (процент пользователей, которые нажали на превью после того, как увидели его)
- retention_rate - удержание (процент видео, который досматривают в среднем)

### Условия отбора:

- ctr > 15 (кликабельность выше 15%)
- retention_rate < 40 (удержание ниже 40%)

### Отчёт включает в себя:
- список названий видео, их CTR и удержание. 
- Отчёт сортируется по убыванию CTR.

Название файлов (может быть несколько) и название отчета передается в виде параметров --files и --report (в нашем случае это clickbait).   Отчёт формируется по всем переданным файлам, а не по каждому отдельно.  
Чтобы сфокусироваться на функционале формирования отчёта и не отвлекаться на рутинные задачи 
(обработку параметров скрипта, чтения файлов и вывод), можно использовать стандартную библиотеку argparse и csv,
а для отображения в консоли tabulate.

## Функциональные требования

- [ ] можно передать пути к файлам через --files
- [ ] можно указать название отчёта через --report (clickbait)
- [ ] в консоль (не в файл) выводится отчёт в виде таблицы
- [ ] отчёт содержит только видео, у которых ctr > 15 и retention_rate < 40
- [ ] в таблице выводятся колонки: title, ctr, retention_rate
- [ ] сортировка по убыванию ctr
